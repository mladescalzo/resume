# React Resume

This repository has been converted to a React app using Vite.

## Run locally

1. Install Node.js if you don't already have it.
2. In the repository root, run:
   ```bash
   npm install
   npm run dev
   ```

## Files

- `index.html` — Vite entry point
- `src/main.jsx` — React app bootstrap
- `src/App.jsx` — Resume content in React components
- `src/index.css` — App styling
- `images/` — Profile image asset
- `package.json` — Project dependencies and scripts
- `vite.config.js` — Vite config

## Notes

The old `script.js` and `style.css` files were removed because the React app now manages behavior and styling through `src/`.
